# Examples

| File | Description |
|------|-------------|
| `basic_usage.py` | Ingest text and query it in ~20 lines |
| `custom_provider.py` | Configure a specific LLM provider (Anthropic, OpenAI, Groq, Ollama) |
| `cli_scripting.sh` | CLI usage: init, ingest, query, JSON output, cache stats |
| `docker_quickstart.sh` | Docker Compose startup with profiles, health checks, curl examples |
